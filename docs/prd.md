# PRD - Claude 提示词模板库

## 1. 产品定位和目标用户

### 1.1 产品定位
**Claude 提示词模板库** 是一个面向中文用户的提示词分享平台，帮助用户更好地使用 Claude AI。

### 1.2 核心价值主张
- 🎯 **快速上手**：新手找到即用的高质量提示词
- 📚 **学习成长**：从模板中学习提示词编写技巧
- 🔍 **精准搜索**：按场景/难度/分类快速定位
- ✨ **一键复制**：点击即复制到剪贴板

### 1.3 目标用户画像

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| **新手探索者** | 刚接触 Claude，不知道怎么写提示词 | 找现成模板，快速出结果 |
| **进阶使用者** | 会用但想提升效率 | 按场景找优化过的模板 |
| **专业创作者** | 有自己的一套提示词 | 分享作品，获得认可 |
| **企业用户** | 团队使用 Claude | 统一管理提示词资产 |

### 1.4 使用场景
- 写小红书/公众号文案
- 生成周报/工作总结
- 分析 Excel 数据
- 写代码/调试
- 翻译/润色
- 头脑风暴/创意生成

## 3. 用户故事

### 3.1 用户故事 1：新手探索
> 作为一个**新手用户**，我想**浏览热门提示词**，以便**快速找到好用的模板开始使用**。

**验收标准：**
- [ ] 首页展示 10+ 个热门提示词卡片
- [ ] 卡片包含：标题、分类、难度、复制次数
- [ ] 点击卡片进入详情页
- [ ] 页面加载时间 < 2 秒

### 3.2 用户故事 2：快速复制
> 作为一个**使用者**，我想**一键复制提示词**，以便**快速粘贴到 Claude 中使用**。

**验收标准：**
- [ ] 详情页有明显的"复制"按钮
- [ ] 点击后提示词内容复制到剪贴板
- [ ] 复制成功显示 Toast 提示
- [ ] 支持快捷键 Ctrl+C / Cmd+C
- [ ] 复制成功率 100%

### 3.3 用户故事 3：精准搜索
> 作为一个**有明确需求的用户**，我想**搜索特定场景的提示词**，以便**快速找到我需要的模板**。

**验收标准：**
- [ ] 首页顶部有搜索框
- [ ] 支持中文关键词搜索
- [ ] 搜索结果按相关性排序
- [ ] 搜索准确率 > 85%
- [ ] 空搜索结果显示友好提示

### 3.4 用户故事 4：模板学习
> 作为一个**学习者**，我想**查看提示词的使用示例**，以便**理解如何正确使用这个模板**。

**验收标准：**
- [ ] 详情页展示提示词原文
- [ ] 展示 1-3 个使用示例（输入 + 输出）
- [ ] 展示使用说明/技巧
- [ ] 示例可单独复制

### 3.5 用户故事 5：移动端使用
> 作为一个**手机用户**，我想**在移动设备上流畅使用**，以便**随时随地查找提示词**。

**验收标准：**
- [ ] 响应式布局适配手机屏幕
- [ ] 触摸友好的按钮尺寸（最小 44px）
- [ ] 移动端导航菜单（汉堡菜单）
- [ ] 页面宽度 <640px 时单列布局

### 4.4 分类体系

| 分类 | Slug | 描述 | 示例 |
|------|------|------|------|
| **写作** | writing | 文案、文章、邮件、报告 | 小红书文案、周报、邮件 |
| **编程** | coding | 代码生成、调试、解释 | Python 脚本、SQL 查询 |
| **分析** | analysis | 数据分析、总结、洞察 | Excel 分析、会议纪要 |
| **创意** | creative | 头脑风暴、创意生成 | 品牌名、广告语、故事 |
| **日常** | daily | 翻译、润色、学习 | 翻译、语法检查、学习 |

### 4.5 难度分级
- 🟢 **初级**：直接复制可用，无需修改
- 🟡 **中级**：需要替换部分变量
- 🔴 **高级**：需要理解结构后自定义
