# UI 设计文档 - Claude 提示词模板库

## 1. 设计原则

- **简洁优先**：内容为主，减少装饰
- **清晰导航**：用户 3 次点击内找到目标
- **一键操作**：复制按钮最显眼
- **移动优先**：从小屏幕开始设计

---

## 2. 配色方案

### 2.1 主色板

| 用途 | 颜色 | Hex | 示例 |
|------|------|-----|------|
| **主色** | Indigo | `#6366F1` | 按钮、链接、激活状态 |
| **主色 Hover** | Indigo Dark | `#4F46E5` | 按钮悬停 |
| **辅助色** | Emerald | `#10B981` | 成功状态、收藏 |
| **强调色** | Amber | `#F59E0B` | 热门标签、评分 |
| **危险色** | Rose | `#F43F5E` | 删除、错误 |

### 2.2 中性色板

| 用途 | 颜色 | Hex |
|------|------|-----|
| **文字 - 主** | Gray 900 | `#111827` |
| **文字 - 次** | Gray 600 | `#4B5563` |
| **文字 - 弱** | Gray 400 | `#9CA3AF` |
| **边框** | Gray 200 | `#E5E7EB` |
| **背景** | White | `#FFFFFF` |
| **背景 - 次** | Gray 50 | `#F9FAFB` |

### 2.3 深色模式

```css
:root.dark {
  --bg-primary: #0F172A;
  --bg-secondary: #1E293B;
  --text-primary: #F1F5F9;
  --text-secondary: #94A3B8;
  --border: #334155;
}
```

## 4. 组件规范

### 4.1 Button 按钮

```
┌─────────────────┐
│   Primary       │  主按钮：Indigo 背景，白色文字
└─────────────────┘

┌─────────────────┐
│   Secondary     │  次按钮：白色背景，Gray 边框
└─────────────────┘

┌─────────────────┐
│   Ghost         │  幽灵按钮：透明背景，文字颜色
└─────────────────┘
```

**规格：**
- 高度：40px（大）、32px（中）、24px（小）
- 圆角：8px
- 内边距：左右 16px
- 最小点击区域：44x44px（移动端）

### 4.2 Card 卡片


**规格：**
- 背景：白色
- 边框：1px Gray 200
- 圆角：12px

- Hover 阴影：`0 4px 12px rgba(0,0,0,0.15)`
- 过渡：`0.2s ease`


- 高度：44px

- 高度：24px
- 圆角：999px（胶囊形）

```
┌─────────────────────────────┐
│ ✅ 已复制到剪贴板！          │
└─────────────────────────────┘
```

**规格：**
- 位置：屏幕底部居中
- 背景：Gray 900（80% 不透明）
- 文字：白色
- 圆角：8px
- 内边距：12px 24px
- 动画：淡入 0.2s，停留 2s，淡出 0.2s

## 6. 关键交互说明

### 6.1 搜索交互

**状态 1：默认**
```
┌─────────────────────────────┐
│ 🔍 搜索提示词...            │
└─────────────────────────────┘
```

**状态 2：输入中（实时建议）**
```
┌─────────────────────────────┐
│ 🔍 小红书...                │
└─────────────────────────────┘
  ┌─────────────────────────┐
  │ 💡 小红书文案生成器      │
  │ 💡 小红书标题助手        │
  │ 💡 小红书标签生成        │
  └─────────────────────────┘
```

**状态 3：空结果**
```
┌─────────────────────────────┐
│ 🔍  xyz123                  │
└─────────────────────────────┘

  😕 没有找到相关提示词
  试试其他关键词，或 [提交新模板]
```

### 6.2 一键复制交互

**点击前：**
```
┌──────────────────┐
│   📋 复制提示词   │
└──────────────────┘
```

**点击后（Toast）：**
```
┌──────────────────┐
│   ✅ 已复制！     │
└──────────────────┘
     ↓
  [Toast 弹出]
┌─────────────────────────┐
│ ✅ 已复制到剪贴板！      │
└─────────────────────────┘
```

**快捷键支持：**
- `Ctrl+C` / `Cmd+C`：复制提示词
- `Ctrl+Enter` / `Cmd+Enter`：提交表单

### 6.3 收藏交互

**未登录状态点击：**
```
[点击 ⭐ 收藏]
     ↓
┌─────────────────────────┐
│  登录后才能收藏哦～      │
│  [去登录]  [取消]        │
└─────────────────────────┘
```

**已登录状态点击：**
```
[点击 ⭐] → ⭐ 填充黄色
     ↓
┌─────────────────────────┐
│ ✅ 已加入收藏夹          │
└─────────────────────────┘
```
